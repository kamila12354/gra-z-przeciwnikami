# Gra z Przeciwnikami

Projekt JavaScript: gra labiryntowa z graczem, monetami, przeciwnikami oraz systemem presetów map.

## Uruchomienie

Najprościej uruchomić projekt przez lokalny serwer:

```bash
make serve
```

Domyślny adres:

```text
http://localhost:8001/
```

Port można zmienić:

```bash
make serve PORT=8002
```

Szybkie sprawdzenie składni JS i pliku JSON:

```bash
make check
```
